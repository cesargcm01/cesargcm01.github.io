# Images Folder

Place your image files in this folder. When you build the app, Vite will automatically serve these files from the root directory.

## Profile Picture
To update your profile picture in the main page:
1. Add your photo to this `images` folder and name it `profile.jpg`.
2. Do not hesitate to use standard formats like `.jpg` or `.png`.
3. If you change the file extension to `.png` or something else, make sure to update the `src` attribute in `src/pages/Home.tsx` to match (e.g., `src="/images/profile.png"`).
